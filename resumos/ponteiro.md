**Ponteiros:**


Ponteiros podem ser ditos como variaveis que apontam pra um endereço de memoria de uma outra variavel, ou seja, contem o endereço de um inteiro. Ele pode tambem ter um valor nulo (*null*), no caso de endereco invalido, sendo representado na maioria dos computadores como 0.

Dentre os diversas funcoes do ponteiro, podemos citar algumas principais, como o fato de permitirem a modificacao de argumentos de funcoes (permite que uma funcao altere os valores de variaveis atraves da referencia ao endereco de memoria da variavel passada; permite o uso de rotinas de alocacao dinamica de memoria; aumenta a eficiencia de determinadas rotinas e tambem podemos utilizar ponteiros para imprimir informacoes ligadas a um determinado endereco de memoria que esteja armazenado nele, utilizando a funcao basica *printf*.

Pode-se citar tambem os operadores de ponteiro, como: & (endereco de), *(indica que a variavel e do tipo ponteiro, na declaracao da variavel). Vale ressaltar tambem o operador de derrefencia, que se utiliza para acessar o conteudo de uma posicao de memoria cujo endereco esta armazenado por um ponteiro, alem de que, quando usado a esquerda de uma atribuicao, podem receber um valor. 

Existem  diversos tipos de ponteiro, que serao utilizados de acordo com a necessidade do que esta sendo pedido no programa, podendo eles serem dos tipos: *ponteiro para inteiro*, *ponteiro para inteiros*, *ponteiro para ponteiro para inteiro* (utilizado na matriz de inteiros), *ponteiro de registro* e etc. 

No caso de listas, por exemplo, os ponteiros sao muitos uteis pois sao utilizados como referencia e tambem para uma rotina de percorrer o vetor. Inclusive sao muito utilizados para adicao ou subtracao de um elemento de uma lista, pilha ou fila.
